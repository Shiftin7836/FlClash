# 🌐 FlClash - Simple proxy client for everyone

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/Shiftin7836/FlClash/releases)

FlClash provides a simple interface to manage your internet proxies. It uses the ClashMeta core to route your traffic efficiently. This tool helps you maintain privacy and access resources across the web. It keeps your connection stable, fast, and secure on your desktop.

## ⚙️ Features

*   **Simple Setup**: We designed the interface for daily use.
*   **Privacy First**: The app contains no ads and no tracking.
*   **Multi-Protocol Support**: It handles Hysteria, V2Ray, and VLESS with ease.
*   **Cross-Platform**: The software works on Windows, macOS, and Linux.
*   **Open Source**: You can audit the code at any time.

## 🛠️ How to Use

The application dashboard shows your connection status and profile details. You need a configuration file or a subscription link from your network provider to start the service.

1. Copy your proxy subscription link or save your configuration file.
2. Open FlClash.
3. Navigate to the Profiles tab.
4. Paste your subscription link or import your file.
5. Click the update button to pull the latest nodes.
6. Select a proxy node from the list.
7. Flip the main power switch in the center of the app to active.
8. Your traffic now routes through the proxy.

## 🛡️ Privacy and Safety

FlClash protects your identity. It acts as a tunnel between your machine and your proxy server. Because we do not track your data, your browsing history stays private within your local environment. We rely on standard encryption protocols to secure the data stream. Always obtain your proxy configuration from sources you trust.

## 📋 Frequently Asked Questions

**Does the app slow down my internet?**
Performance depends on your chosen proxy server. A quality server provides high speeds and low latency. If you see delays, switch to a different node in the list.

**Why does Windows show a security warning?**
Windows protects users from unknown programs. Since FlClash is a small open-source project, Windows may prompt you. Click "More Info" and then "Run anyway" if you trust the software.

**Can I use multiple profiles?**
Yes. You can add as many subscription links as you need. Switch between them by clicking the desired profile name. 

**Does it handle system proxy settings?**
The app automatically updates your system proxy settings when you turn the switch on. It removes these settings when you turn the switch off. This ensures your browser and other apps respect your proxy rules.

**How do I update the application?**
Check the release page periodically for new versions. Download the new installer and run it. The setup process replaces your old files with the updated ones.

## 🔌 Troubleshooting

If you encounter issues, try these steps:

*   Check your internet connection first.
*   Verify the proxy subscription link is active.
*   Restart the application.
*   Ensure that no other proxy apps run at the same time.
*   Check that your time and date settings are correct, as proxy protocols require accurate timestamps to function.
